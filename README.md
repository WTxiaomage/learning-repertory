# learning-repository - 学习仓库

---

- 这是一个学习仓库，里面记录自己的学习经验和想法。
  <br>

## 一、初衷

- 一直以来都想建一个自己学习的经验和想法的完善笔记，但是很多时候疲于奔命，很少停下来反思，最近因为发现自己曾经学过的东西很多好的想法都忘记了，痛定思痛，决定从现在开始慢慢累积，把自己学到的点点滴滴都记录下来，希望看着这个仓库越来越繁荣，就会有一定的成就感。
